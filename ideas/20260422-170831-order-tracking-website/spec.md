# Customer Order Tracking Website

**Status**: draft  
**Created**: 2026-04-22  
**Author**: Ray Carroll

## Overview

This idea addresses the problem that customers currently must call customer service to check their order status, which is time-consuming for both customers and support staff. The solution is to build a self-service website where customers can track their order status in real-time without needing to contact customer service.

## Problem Statement

Users currently check order status by calling customer service. This creates several issues:
- Customers experience long wait times (20+ minutes on hold)
- Support agents spend significant time on simple status inquiries
- Support capacity is consumed by routine questions rather than complex issues
- No 24/7 availability for status checks

This problem has been encountered and is a problem that needs to be solved to improve customer experience and support team efficiency.

## Proposed Solution

Build a website where customers can track their order status in real-time without calling customer service. The website will provide:
- Self-service order status lookup
- Real-time updates from the order management system
- 24/7 availability
- Detailed order information and estimated delivery dates

## User Stories

1. **As a customer**, I want to see my order status instantly, so that I don't have to wait on hold for 20 minutes.

2. **As a customer**, I want to see estimated delivery dates, so that I can plan to be home when my package arrives.

3. **As a support agent**, I want fewer status inquiry calls, so that I can help customers with more complex issues.

## Technical Approach

Must integrate with existing order management system (SAP). The technical implementation will need to:
- Connect to SAP APIs for order data retrieval
- Implement customer authentication
- Design responsive web interface for mobile and desktop
- Ensure real-time or near-real-time data synchronization
- Handle SAP system unavailability gracefully

## Alternatives Considered

We considered upgrading the phone system with automated status responses, but a website provides more detailed information and is available 24/7. The website approach offers:
- More comprehensive order details
- Visual tracking information
- Better customer control over when and how they check status
- Reduced telecommunication costs
- Better scalability
